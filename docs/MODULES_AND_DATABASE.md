# Functional Coverage

## Authentication and Roles
Customer-facing registration creates an EMPLOYEE user profile. Administrative users manage roles and account status.

## Attendance Workflow
Employees clock in and out. The system derives late, undertime and overtime minutes from the assigned shift template. Admin can manually record absence/leave/holiday status and review correction requests.

## Leave and Overtime Workflow
Employees submit requests. Administrator approves or rejects them with remarks. Triggers automatically create user notifications when workflow statuses change.

## Reporting
Views and CSV export cover roster, attendance, late/absence summary, leave monitoring, overtime monitoring and payroll attendance summaries.
